                        

William Porter。图中为5岁，与每一SCP-4662基因完全匹配。

项目编号 : SCP-4662

项目等级 : Euclid

特殊收容措施 : 除SCP-4662-11外的所有现存SCP-4662个体将收容于长期冷冻存储内，直至伦理委员会达成对合理收容的一致意见。SCP-4662-11将依照附录4662-11中所述加以收容。 Arthur与Maureen Porter将被收容于Site-77的标准人形收容间内，对任何SCP-4662显现应立即予以报告并收容。

描述 : SCP-4662是影响密歇根州迪尔伯恩一对夫妇Arthur与Maureen Porter的异常现象。在SCP-4662发生期间，Porter夫妇之子William的一个复制体 (称为SCP-4662个体)将随机显现在Arthur或Maureen身边六千米范围内的某处。每个SCP-4662个体都年约五岁，在基因上与William Porter相同，且拥有其原有过往生活的记忆。采访Porter夫妇后确认这些记忆与真正的William Porter相一致。在显现后，个体会表达出回家的强烈愿望，且本能地知晓其父母所在地；此二特点会在约10小时内逐渐消失。

SCP-4662与某种可能的次要效应存在强关联性，每一个体的显现都对应着六公里效应区内的一名儿童失踪，实际效果上是该个体将其替代。这些儿童均为4到7岁；没有其他明显的选择标准。若SCP-4662显现时没有这样的儿童存在，个体将会出现在某一完全随机的地点。基于伦理考虑，对此性质尚未进行完整测试。

附录4662-a : 收容历史

SCP-4662显现的首次纪录于2003年1月22日发生于迪尔伯恩，有一SCP-4662个体被发现试图逃离█████与███████ Shah家中，而此前不久Shah家4岁的女儿失踪。基金会被告知此事，将该个体和Arthur, Maureen以及当时12岁的William Porter带入扣留。采访时，三名Porter都对此现象表现出一致的困惑。

由于SCP-4662个体没有表现出其他异常性质，决定继续将其收容1但施以记忆删除，Porter一家被释放，对其严密追踪监控是否有后续异常现象。

第二次SCP-4662显现记录发生于2007年7月3日，有一名个体在另一户迪尔伯恩家庭的孩子失踪后于其家中被捕。和之前一样，Porter家三人和该个体都被带入基金会扣留，此时发现Arthur与Maureen的供词几乎是对前次供述的逐字重复。

对Porter家住所的更多调查后，发现其后院内掩埋了多具人类遗骸。发掘后发现六具遗体，均为年约5岁，基因上均与William Porter匹配，均明确死于他杀，且大致死亡时间均与当地一名儿童的失踪日期相对应。其中两人似乎是在2003年到2007年间被杀，最早的尸体于1996年被杀，是唯一能与William Porter的1991年生日相一致的个体。该尸体也有独特的死亡状况；其他个体都是死于颈部相对有效的伤口2，而最早的这具尸体死于一系列胡乱的钝力创伤。

附录4662-b : 采访抄录， 07/05/2007

对象 : Arthur Porter, Maureen Porter

讯问者 : Dr. Mangala Shastry

Dr. Shastry : 开始记录。Porter先生与女士，我想就最近围绕你们家发生的事件问些信息。

A. Porter : 我们已经说过了。我们完全不知道你们找到的这孩子是谁，或者为什么他看着像Billy。你们还要把同样的问题问到多久？

Dr. Shastry : 嗯。你们想看看这些照片么？我们昨天在你家后院拍的。

M. Porter : 操。

A. Porter : 一个字别说，Maureen。别他妈再说一个字。

15秒沉默，两名对象查验照片；他们的查验草率且大体无情绪反应。

Dr. Shastry : 考虑到这份新证据，你们想修正下之前的供述么？

A. Porter : 不。我知道这怎么回事，我要律师。我有权，你知道的-正当程序之类的，我知道我有权利。

Dr. Shastry : “正当程序”？你最近有看过周围么？这里看起来像是警察局？Porter先生，我们只要想就可以把你们一辈子都留在这。你们对此配合能帮助我们考虑这是否是不必要的举措。

10秒沉默。

Dr. Shastry : 我们从最基本的问题开始：是否是你们造成了这些，呃，你们儿子的克隆人存在？你们是不是在把邻居的孩子变成Billy的复制品？

M. Porter : 不是！当然不是，我们从来没要求-

A. Porter : 该死，Maureen。闭嘴。

Dr. Shastry : 别担心，我们不会猜有自动机器和兼职女仆在运营地下克隆工厂。在这些复制体第一次出现前，你们能不能想到有什么反常事件？

A. Porter : 你是问我们是不是骂了个吉普赛女人或者惊扰了印第安人墓地之类的？嗯，没那么走运，女士。

Dr. Shastry : 那么就是说它们就这么突然开始出现了。为什么你们认为必须要杀死它们？

30秒沉默。

Dr. Shastry : 好吧，让我说下我们认为的时间线。你们的儿子，William Porter，出生在1991年1月3日；档案里的出生证明似乎是真实的。五年后，因为某种意外钝性创伤和刻意疏忽，你们让你们的儿子死去了。这之后不久-大概是在24小时内，毕竟没人注意到他失踪过-你们儿子的第一个复制体出现，也就是你们过去二十年里养大的那位。你们肯定想着这是奇迹，是重头开始的机会直到下一个复制体出现。除非你们自己交代，对你们真正的儿子到底出了什么事只能这样才能合理解释，所以你们解-

A. Porter : 天啊，你们都知道了-我是说，你们都这么肯定了，为什么还要我们？你们到底要图我们些什么？

Dr. Shastry : 说实话么？到这会儿所有的事情都一目了然，只有一个问题：你们为什么要做这些？William - Billy -有受多次身体和情绪虐待的迹象，我们对他的所有采访都指出，他的家庭生活里完全缺乏任何形式的喜爱。别说是爱他，你们甚至好像都不怎么待见这孩子。所以，再说一次，为什么？

M. Porter : 你们可以让我们忘事，对吧？

Dr. Shastry : 什么？

M. Porter : 像有部汤姆·李·琼斯电影，你们可以对着我们的眼睛闪一下我们就都忘了？只是我一直在想几年前失踪的那个Shah家孩子，我记得那时候在想我们家Billy是不是与此有关好吧，他就是，对吧？你看，我们没你们以为的那么傻。

Dr. Shastry : 我不明白你在说什么。

M. Porter : 你们能不能就让我们把这些都忘了？所有的。整件事情。

Dr. Shastry : 十一年的全部记忆？

M. Porter : 对。

A. Porter : 她是对的，这样就最好。

M. Porter : 反正没什么值得留下的。

Dr. Shastry : 采访结束。
附加记录：

采访后，已收容的个体被重分级为SCP-4662-5与SCP-4662-8，“William Porter”被重分级为SCP-4662-1加以收容， Arthur和Maureen Porter被安置于无限期收容中。不批准使用任何记忆删除。

附录4662-c : 长期实验结果

截止09/21/2009，又有两名SCP-4662个体显现，针对SCP-4662效应的长期后果提起了严肃的伦理顾虑，特别是基金会对此类产生个体负有何种责任。在多次解决此问题失败后，对初级研究员Alan Covington提出的SCP-4662无效化尝试提案予以采纳。

根据提案，下一个SCP-4662个体在显现后被立即收容、隔离，以Arthur与Maureen Porter已死于意外为掩盖进行抚养，且该个体—允许其自认为是真正的William Porter—患有某种需要长期隔离的传染病，必需接受持续收容。在其他所有方面，该个体将在初级研究员Covington的直接监护下、依照广为接受的儿童养育心理学加以抚养，学业和娱乐活动亦由其负责监督。这些技术令其在心理测试基准上相比于其他SCP-4662个体有显著提升。

在测试实施后，迄今不再有SCP-4662个体显现。将SCP-4662重分级为无效化的提议正在审议中。

脚注
1. 当时仅作为一次异常现象录入
2. 可辨识出所有尸体的颈动脉戳刺伤口，且未发生严重分解
