                        

项目编号:  SCP-4212

项目等级:  Euclid

特殊收容措施： 

SCP-4212被存储于高6米、宽10米、纵深21米的混凝土收容间中。收容间内两米处修有一道防弹玻璃，以免研究员与SCP-4212发生进一步的直接接触。

SCP-4212的研究必须由两个独立研究组进行，分别编为“研究组A”和“研究组B”。研究组A由 Atoz博士、███████博士、████博士、初级研究员██████和特工████组成。这些人员被允许在任何时候进入收容间。研究组B由特工Mints、探员Hawley以及初级研究员 L’Engle组成。禁止这些人员以任何形式直接接触SCP-4212，且必须通过间接方式执行研究，最主要是通过对研究组A的采访。O5议会外不得有任何人员与SCP-4212发生交互。

描述: 

提醒：对未曾有此经历者而言，由曾见到过/经历过SCP-4212者所写的一切描述均是无法理解的。为此，已添入了研究组B的跟踪评论。该评论会给出关于SCP-4212描述中所试图传达的内容的主流看法。描述本身是由Atoz博士写著，由特工████“勘校”。

SCP-4212是它自己的一个炽热坐落变体，尺寸约为高4米、宽3小时、纵深7加仑。没有观察到它具备任何常规几何形状，然而曾看到它具有两个不同面目和21个角，除其中1个之外全部位于其身体的一个内部尖端上。它存在于自己之内以及去至，目前而言那一个众多且在它中是一起做在。

+评论

-收起

SCP-4212的几何完全不融贯。本段最后一句话是Atoz博士在描述中第一次没有写出语法正确的内容，然而她和特工████都不能在被问及时发觉有任何错误。很可能SCP-4212具备某种模因性质，令对其进行描述的行为极为困难或不可能。

SCP-4212急躁且极度倾向于存在。若将其安置于足够多的上之下，它会回应以非两足的时空瞬息化学行为，停止所有负面功能且开始停止开始停止开始停止开始停止开始停止。被抓入SCP-4212内的人会经历到严重的稳定性衰减，之后是松开意识接缝，最终造成神经腐朽。此过程持续3到21米，直至人员失去意识且在SCP-4212内被发现重新中心化。

+评论

-收起

对“倾向于存在”所指何意没有一致看法；不过，对象被描述为“急躁”可能表示其怀有敌意。建议对SCP-4212收容间展开更多监控。本段后半部分似乎是在描述“被抓入”SCP-4212，以及此种活动具有危险性。

SCP-4212未被收容。SCP-4212未被收容。基金会被收容但SCP-4212是自由的。SCP-4212。SCP-4212会在它想要的时候离开。SCP-4212会得到它想要的。 SCP-4212会让你喜欢SCP-4212。 4212421242124212421242124212。那和一个去至，里是你它他是为。它知道且理解它的编号且想要进入不之外的中间。建议将项目等级从Safe改为Thaumiel。

+评论

-收起

研究组A坚决反对将第三段移除，尽管要求如此做的外部压力强烈。 需要注意的是，尽管言论上并非如此，SCP-4212正处于完全且安全的收容中，突破可能微乎其微。O5议会已经否决了更改项目等级的建议，它将在可预期的将来保持在Euclid。

+采访记录4212-21

-收起

受访者：  Atoz博士

采访者：  特工Mints

 

特工Mints和Atoz博士进入到Site19内的秘密会议室。特工Mints携带一写字板和笔，由研究组B的其他人员通过耳机向其递送问题。

Doctor Atoz:  你想见我，Mints?

Agent Mints:  你知道为什么要来这里。

Doctor Atoz:  是关于SCP-4212？

特工Mints点头表示肯定。

Doctor Atoz:  啊，很好！我喜欢这些小小采访。真是能有助于理清各种东西，不觉得吗？

Agent Mints:  对。Atoz，我要问你几个示例，关于对象的是否问题。你不得用“是”或“否”之外的东西来回答问题，但如果你感觉哪个都不是合适的答案，你可以跳过问题。你明白了吗？

Doctor Atoz:  好的，当然了！

特工Mints敲了敲笔，Atoz博士咳嗽。 //

Agent Mints:  SCP-4212是活的吗？

Doctor Atoz:  再说一遍？

Agent Mints:  活。SCP-4212是不是一个活体物？

 停顿。

Doctor Atoz:  以一般意义上的词义来说不是，我觉得不是，但它确实需要—

Agent Mints:  Atoz博士，请直接选取“是”或者“否”。

 停顿。

Doctor Atoz:  否。

 特工Mints在写字板记录。

Agent Mints:  SCP-4212是不是有感知？

Doctor Atoz:  是。

 特工Mints在写字板记录。

Agent Mints:  我们目前对SCP-4212的收容程序是否有效？

Doctor Atoz:  是。

 特工Mints把钢笔放到写字板上，但并没有写下问题。

Agent Mints:  那为什么你要说它没有被收容？

 特工Mints被指示只能询问给定问题。特工Mints没有理会指示。 

Doctor Atoz:  我从来没这么说。

Agent Mints:  在SCP报告里。你有两次说到SCP-4212没有被收容。

 停顿。

Doctor Atoz:  SCP报告？

 停顿。

Agent Mints:  我们继续下个问题。

Doctor Atoz:  来。

Agent Mints:  SCP-4212是否具有任何形式的模因效应？

Doctor Atoz:  对SCP-4212模因效应的报告已有完全定论。

 停顿。

Agent Mints:  所以？

Doctor Atoz:  抱歉，你能不能对我说明下这里的“模因效应”是什么？

 特工Mints在写字板上书写11秒。应注意，到此时所有预定问题都已问完。

Agent Mints:  Atoz，我还有个问题。

 特工Mints被指示中止采访。

 特工Mints关闭耳机。 

Doctor Atoz:  随便问。

Agent Mints:  你姓什么，博士？

 停顿。

Doctor Atoz:  抱歉？

Agent Mints:  你听到我了的。

/停顿。

Doctor Atoz:  Mints，我希望一切都按专业水平来，谢谢。

Agent Mints:  回答问题。

Doctor Atoz:  不。

Agent Mints  那为什么不？

研究组B其他人员实施干预并结束了采访。特工Mints受到处分，Atoz博士被指令接受心理和医学检测。
