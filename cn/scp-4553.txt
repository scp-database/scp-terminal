基金会政治控制管理将维持一套主动和被动结合的反制措施，以防止SCP-4553被选举到美国众议院中。这些行动包括对SCP-4553竞争者的宣传资金捐助、媒体屏蔽、阻挠以及以破坏SCP-4553公众声望为目标的媒体编造。参见文件“紧急事态4553-夜爬者”获取策略部署完整列表。

描述 : SCP-4553是一团具有智能的拟人态蠕虫群，正在积极竞选美国国会议员职位。

SCP-4553自称为“Bob Shepherd”，年龄42岁，自称来自内布拉斯加州的一座小镇瓦伦丁。 尽管没有发声器官，它仍拥有说话能力，初步估测其智力略高于人类平均水平。SCP-4553曾多次自愿地提供身份文件，以证明其拥有美利坚合众国合法公民地位和内布拉斯加州居民身份，包括：

出生证明1
纳税记录（无违法）
驾照
护照
卡尼县内布拉斯加州立大学政治科学硕士学位2

此外还有其他证件，除提供者实为一团蠕虫构成的非人实体外，证件中没有明显的矛盾之处。

SCP-4553正作为独立候选人公开参选国会职位，截止本文件创立时已在投票中获得了89.0%的支持。在内布拉斯加州第三选区出现此种压倒性支持态势是值得注意的，因为该区本是全国最主要的共和党选区之一，上次选出非共和党候选人是在1961年。因此，怀疑SCP-4553具有某种模因性强迫效应，能对选民心理产生影响，然而目前尚未就此进行过决定性验证。

这种假想性的模因场并未延伸到SCP-4553的外形上，其（在公众和官方层面）始终是一团86千克的蠕虫群，大致聚合为一人类男性形状，一般穿着三件套西装配绿领带。SCP-4553的目标选民、同事、反对者及所有好友都能辨认出“Bob Shepherd”是一团蠕虫，且不认为这具有任何负面性或相关性。支持者和批判者都将SCP-4553视作人类一样对待，同时也能观察并理解到其蠕虫形状的性质，但没有因这种矛盾出现一般会有的明显认知失调。基金会曾采访过多名内布拉斯加州公民以试图研究SCP-4553对人类心智的影响。摘录如下。

日期 : 2018年7月22日

媒体来源 : 基金会前台组织进行的采访视频抄录，伪装为内布拉斯加州北普拉特的一所独立政治研究委员会来进行。

采访者 : 基金会调查特工Anthony Rawlins

受访者 : Randall Fischer，47岁，汽车机械师，SCP-4553竞选的支持者

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Agent Rawlins : 感谢您来与我们对话，Fischer先生。

(Fischer在Rawlins桌对面的座位上就坐。两人握手。)

Fischer : 别在意，朋友，我很乐意。我很愿意参加这个程序。信里说这是什么政治调查之类的，对吧？

Agent Rawlins : 对，没有错。不会超过二十分钟，这也能帮我们更好的理解这边的选举是怎么进行的。快速无痛。还有免费咖啡和甜圈作为额外福利，哈哈。

Fischer : (发笑)对对，你们前厅那边现在损失惨重。把我老Randy摆在一碟子甜圈前，没有一场大扫荡是不可能的。

(Rawlins大笑，然后从抽屉中拿出笔记版和笔。)

Agent Rawlins : 好的。首先，我想问你些关于你个人政治经历的基本问题，好吗？

Fischer :没问题，随便问。

Agent Rawlins : 首先，你有登记加入任何政党吗？

Fischer : 有，共和党。

Agent Rawlins : 你曾经投票给不属于共和党的候选人吗？

Fischer : 没有先生。我的血天生是红党。

Agent Rawlins : 你觉得你这一生一共支持过多少位公共职务候选人？国家级，州级，地方，全都算。大概估算下。

Fischer : 呃三十多年了。不管到底是多少。我猜大概得有百来人了。

Agent Rawlins : 真是非常惊人，Fischer先生。那么我想您也计划要参加本选区的下一轮国会选举？

Fischer : 靠当然必须的！ 现在要休息太晚了，哈哈。

Agent Rawlins ：您是属于第三选区，对吧？

Fischer : 对。

Agent Rawlins : 您有决定好要支持哪一位候选人了么？

Fischer : 怎么，当然，一开始就定了。我要投给Bob Shepherd，百分百肯定。他就是我们需要的人。

Agent Rawlins : 麻烦您确认下我没搞错，这个Shepherd可是列为独立候选人的，对吧？不是共和党？

Fischer : 哎嘿，没有人是完美的吧，哈哈。对，他不是共和党人，但要我说实话，他大概是最接近真正共和党候选人的人了，哪怕他举的不是一个颜色。上次那位就是个该死的污点，我很遗憾的说，其实从那之后我对这边的共和党有点丧失信心。都是些性丑闻之类的破事。恶心。不敢相信我居然投给了这种变态。然后他们推出来顶替他的那位也好不到哪去，就是个奥马哈来的瘦麻杆律师。但是Bob？好吧，他不是共和党的，但我觉得他一样是国家需要的人，让标签见鬼去。

Agent Rawlins :我明白了。好吧，给我推荐下他如何。Bob Shepherd为什么这么合你意呢？

Fischer : 该死，要从哪开始呢，我完全可以叨叨几小时，但我精炼点为好。第一，他有个好头脑。看起来是有点紫不啦叽湿乎乎的，对，但还是很好。我看过他提议的下个阶段税务计划，真的很有道理。我们到最后是得稍微掏下腰包，这肯定的，但这哥们说的很对，这边的东西得修整了。到处都是漏洞、垮桥，等等。卫生部门完全就是一坨狗屎，可以这么说。一点不安全。他这个“现在修复，一劳永逸”的政策你完全没法反驳。 是时候得有人把基础设施当讨论点提出来了。上面的那帮泡沫脑袋红莓子整天说什么检修桥梁更换管道，但他们没一个人真正做了实事，他们只是想着中选然后连任。Bob是真正定了计划的，我看到了。

第二，你就是止不住要喜欢这人。他是个好的道德榜样。个人记录里没一丝污点。他很正派。在市政工程岗位上干了好多年，把所有时间都投在了基础服务改革上。也取得了些重大胜利，让他邻里的生活安全多了。回头看看他在基建、税收、侵权法改革、污染、打击犯罪上的观点，他没有哪怕一次出尔反尔过。他始终如一。还有，他从来没被人逮到撒过谎，一次都没。我这么说一点不骄傲，但我确实曾经投给明知是撒过谎的人，因为没有更好的选择。现在，我可以真真的说，我有信心对上帝发誓，这次我可以给一个真正可信的政客投票。我从来没想过居然有天能见得到！这就是Bob。诚实。同情，而且明智。 你对候选人所要求的一切他都有。

(Rawlins沉默一会儿，记笔记。)

Agent Rawlins : 好吧，Fischer先生，既然您都这么说了，也很难再反驳些什么。那么，您有意识到Bob Shepherd其实，非常明显，是一坨穿着衣服的人形蠕虫，对吧？ 这个人是呃，一坨蠕虫。他完全是由蠕虫组成的。

(Fischer皱眉将手交叠。)

Fischer : 好吧，对对，但是其实并没有嘴，我是没看到有。

Agent Rawlins : Bob Shepherd不是人类这事完全没有让您困扰？

(Fischer面露愠色，在椅子上向前倾身，用手指着Rawlins)

Fischer : 你看好了，小子。我已经厌烦了你这些旁敲侧击。就因为我是共和党，我没投票给那娘娘腔城市仔奥巴马，可不代表我是种族主义。我没这么狭隘。我关心的是一个人的思想和行动，不是他的肉体。我明白Bob Shepherd和我们其他人是有点不一样，但你说“不是人类”？这不只是歧视了，这是错误。Bob Shepherd是忠诚的内布拉斯加人以及，凭上帝由我见证，一位优秀的美国人，不管他是黑是白，是蠕虫是图钉还是他妈的巧克力布丁！我也不会陪你在这灌输什么他不是人了。听听你自己的话吧，上帝！

(Rawlins特工吞咽一口，点点头，记笔记。)

Agent Rawlins : 是的。这有点呃，不像话。我道歉。

(Fischer靠回椅子上，再次把手交叠。)

Fischer : 太他妈对了！现在赶快把剩下的问题念完，老子要挪屁股远离你们在这搞的劣质智库。还有我走的时候别指望能有一个甜圈剩的下来，想都别想。

物理收容SCP-4553被证实不具可行性，因该实体还具备第二异常效应。若将其从天然环境带入基金会监管，SCP-4553将失去异常性质，解体为无组织性的活体蠕虫。3然后它会在内布拉斯加第三国会选区内的另一地点重新出现。严重物理破坏，诸如使用高能对人武器实施刺杀，会造成类似结果，令SCP-4553解体并重现于别处。 SCP-4553没有对这些身体破坏事件表现出记忆，其发生过程从未被直接观察到。

SCP-4553采用的人格面貌“Bob Shepherd”在法律和社会层面上始终一致。虽有异常的物理外形，SCP-4553 并未表现出反常或危害性行为。“Bob Shepherd”的个人历史从其“出生”于1978年开始有清楚记载，尽管其明显具蠕虫外貌，在个人经历内容上与其他表现良好的美国公民并无差异。它的竞选纲要（始终包含发展基础设施、重振个人的宪法自由权利、制度性反腐败提议和环境保护）始终一致，并未对美国社会或共认常态整体呈现出严重威胁。

附录4553-01 : 竞选演说摘录，2020年8月3日

(SCP-4553站在礼堂后方的舞台上，调整话筒，对人群举起一只手，现场人数约1000人。观众欢呼，然后静默。)

SCP-4553 : 早安，各位。

(SCP-4553看向天空，举手挡住照射面部的阳光（尽管它并没有眼），然后将注意力回到人群。)

SCP-4553 : 各位。今天是个美好的日子，不是吗？我肯定你们都不愿意听我在这瞎抱怨，所以我还是精简明快点为好。

(实体看向礼堂，沉默了一会儿，然后随意地把它带来的讲稿甩了出去。人群大笑。)

SCP-4553 : 我的竞选助理觉得这样很合适，这样给我几页纸念就好，但我不想在这待个三小时，你们也不想。所以这次我为何不单飞一把呢？抱歉了，Bill。下次我把你的讲稿念两遍好了，你看如何？

(Shepherd对着它的竞选助理William O'Connor挥手，他此时站在舞台一边。O'Connor挥手回敬，微笑着摇头。人群大笑，略有掌声。)

SCP-4553 : 所以。我的美国同胞们。一般都这么开头的，对吧？我的美国同胞们。其实，我觉得这太多余。我和你们不少人是一起长大的。靠，前面那位不就是Tom Hammerstein么。十一年级的科学课，记得吗，Tom？

(Shepherd把手举到头边做神秘状，四处张望，靠向话筒)

SCP-4553 : 我其实考试有抄过你好几次，哥们。你配平化学方程总是比我好很多。不过可别往外传–如果这事流出去，我的竞选就得凉了。

(笑声)

SCP-4553 : 总之吧。重点在于，我觉得不需要把你们都说成是好吧。和别人在一个岗位干十五年还面带微笑的原因是一样的。因为我在乎。我一辈子里认识了你们很多人，你们可以赌我从来没忘记过这个事实。正是这种羁绊的感觉，爱的感觉，在这么多年后把我带到了这个地方。

(停顿)

SCP-4553 : 有人问我为什么要来这边浪费钱，屁颠屁颠忙选举。政客的报应明明比地狱烈火还惨，哈哈。 好吧，答案是因为你们所有人。你们可曾感觉到自己可以做的更多？像是你还没有把身心物尽其用？以前我这么感觉过很多次。不是说我有什么不开心的。一点都不。我夫人不会让我这样的，嘿，亲爱的！

SCP-4553对着它的妻子、39岁的Sandra Shepherd招手，她和两名非异常儿童一起坐在观众前排。她对着人群大笑，挥手致意。

SCP-4553 : 她就是我生命中的明灯，各位。偶尔也是赶牛棒。他们说每个强大的男人背后都有个更强大的女人，这绝对是福音。她给我说敢这么说就打死我。

(大笑)

SCP-4553 : 随便吧。我感觉我还不满足。我可以做更多的事情。很多人都有这感觉，有时候，我觉得是。所以我决定搞什么鬼，我们说干就干。我要对你们敞开胸怀，各位，有很多事情需要去做。现在，你们以前听我说这件事很多次了，所以我要为新来的各位提供一些要点。

好。烦扰我的事情一号—基础建设。对。对，我知道，哪怕是“基础建设”这个词都让人感觉昏昏欲睡，但是听我说完。各位，我们的国家是被裂缝分割的平原。我说的是字面意义。我们的桥梁百分之八十满是锈迹，建筑上的霉菌比那蓝起司还要多, 道路上满是坑洞，比那什么还多来着，呃就是另外一种起司。艾森豪威尔把大部分这些东西修起来，但自此之后根本没人照看。艾氏是位伟大的总统，伟大的美国人，但他这么做也是五十年前的事了，我们亟待一些翻修整新，全国上下。你们是否知道，在2013年美国土木工程师协会对全国的基建质量进行过一次调查，为我们的作为打了一个字母评级？你们知道成绩是多少吗？D+。D+!朋友们，我可能就是一坨粗陋的乡村蠕虫，但就算是我也知道这太悲哀了点。这不只是尴尬，这是公共安全危害。我们需要有资金和聪明的男女人手，去修复我们的管道电线等等等等，不然在我们就得在带电的大粪里埋头苦干。

说道恶心事，麻烦二号来了–环境。那什么，我不是那种生态狂热者，非得要你们全都啃萝卜、为大地母亲痛哭流涕之类，但是拜托。我们对着空气和水源排放的东西根本就是不可接受的。我们对大公司用恶心化学品污染我们的后院太过手软。我不想要一瓶冰镇六价铬配特氟龙，我要的是水。好吧，我其实想要的是啤酒，但水是最接近的了。我爸，上帝愿他安息，是个单纯的人，他以前说过一句话：“不要在睡觉的地方拉屎。” 原谅他的法国腔，哈哈。我不知道你们是怎样，但我觉得这话有道理。我们必须找到更多办法负责任地处理垃圾。我想要一个清洁的美国，我保证要让大公司们对此开诚布公。

然后就是第三件事—人尽皆知的不良行为。既在政府内，也在美国的百姓间。我不知道过去这二十年发生了什么，但靠该死的好像人们都不愿意再用尊重和敬意彼此相待了。你们看到国会山的政客从巨富那里拿钱办事，让他们靠践踏弱小壮大自己。你们看到有些人大摇大摆走来走去，心头没有一丝人类的团结，凭肤色和体型便低看他人。这是宗派主义，这是排他主义，这是彻底的不美国。

我信仰着一个欢迎所有人的美国，无论他们长相如何，管子向哪边。一个能为弱者抵御邪恶的美国，没有细则条件。自由女神像上面写的清清楚楚：“送给我，你受穷受累的人们，你那拥挤着渴望呼吸自由的大众。”它可没说，“只送给我有特定生理结构和意识形态的人们。”碑上的那首诗叫做《新巨人》，这就是我希望的美国。一个巨人。不是霸凌、暴力或仇恨的巨人，而是自由的巨人，和平的巨人。 我想要我们高举火炬，告诉所有人若我们团结奋斗会有怎样的前景，无论我们是何外貌、来自何方。我想要我们成为人类的榜样。一道明亮耀眼的光芒，打破一个如此、如此黑暗的世界。

现在，你们有些人把这叫理想主义。多愁善感。不切实际。但是你们知道吗？我把这叫尊重。我把这叫团结。我把这叫做一个真正的美国人。

(起身鼓掌，雷鸣般的喝彩声)

目前，基金会对SCP-4553竞选开展的所有阻碍、混淆和污名化工作均未成功。尽管是共和党主导选区的独立候选人及一团非人类蠕虫，“Bob Shepherd”仍在当前投票中拥有89%的支持。早期基金会前台组织曾尝试基于其明显的异常性质来污名SCP-4553，却遭遇了压倒性失败：绝大多数内布拉斯加州投票民众将此举谴责为“偏执顽固”或“种族主义”。参见文件“紧急事态4553-夜爬者”了解在SCP-4553当选众议院（似乎已无可避免）后的最佳后续行动。
