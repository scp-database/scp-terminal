                        

SCP-4111内部某洞穴。

项目编号:  SCP-4111

项目等级:  Euclid

特殊收容措施:  要制造SCP-4111-1个体，需获得三级以上权限。任何存在但尚未投入使用的SCP-4111-1个体必须被储存在安全的异常项目收容室中，除非向SCP-4111转移的行动被批准，否则不允许接触盐水溶液。

希望长时间在SCP-4111中探索的基金会成员需配备标准干旱气候紧急逃生背包。出于任何原因被困入SCP-4111的工作人员都应尽快被找回。

在SCP-4111内发现的任何其他人员将被回收，在被采访后接受记忆清除并重新回归社会。

描述:  SCP-4111是一个可通过特殊手段抵达的位置，目前被认为不在地球上1 。 该手段能够将多名人员（或物体）转移至特定环境下的某一区域 (详见 SCP-4111-1说明)。

SCP-4111整体位于地下，有可供呼吸的气体环境，表层土地多为凝固岩盐2，偶有高热熔融岩盐形成河流、湖泊与大片海洋。除熔融岩盐发出的光外没有其它可见光源，且进一步调查显示此地存在一高度未知的天顶，同样由岩盐构成。所有SCP-4111中的自然结构均为立方体结构，目前尚未在其中发现天然形成的弧线。

一个未激活的SCP-4111-1个体。

SCP-4111-1被用于将实验员或物体传送至SCP-4111。制作SCP-4111-1所用客体不必完全相同，但其填充物与形状需遵循随PoI-418被发现的指南之规定，该指南由Gemenskap Coalition组织4（包括该SCP-4111-1个体）均将被传送至SCP-4111。

唯一从 SCP-4111 返回的方法是死亡。触发SCP-4111-1进入SCP-4111的实验员如在任何情况下死亡，将会与其直接接触的人员和物体共同传送回触发前之位置。实验员返回后，所有在SCP-4111内产生的伤病皆会消失，SCP-4111-1 个体也将出现在实验员附近。实验员将保持在SCP-4111内部活动及自身死亡的记忆。

勘探记录中发现了若干疑似不属于SCP-4111的物品，详见下方清单。该清单表明，在基金会发现SCP-4111前，已有人员多次进入SCP-4111并对其进行了探索。

发现:  在9/28/20██，首个已知曾经进入过SCP-4111的测试者PoI-418，突然出现在2009年前后沉没于大西洋的████████残骸附近。 PoI-418的图像来自该海域一水下机器人。报道称，一男子在此次海域探测过程中突然出现，基金会获悉后立刻清除了相关目击者记忆并封锁消息。PoI-418的图像被封锁，记录该图像的水下机器人被销毁后沉入海中，基金会对外宣称这是由于探测过程中的机器故障。

PoI-418:  真是个好姑娘、一匹好马。你怎么样？

队伍中其余人放慢脚步，谨慎地靠近果园和 PoI-418-H。Agent Pele把面具和头盔递给 PoI-418。

Agent Pele:  你该庆幸这里的空气可以呼吸，418。难以想象……明明有大量火山中的气体排入空气……

PoI-418:  我得让她知道是我！我们戴着面具看上去都一样。

Agent Pele自言自语。

Agent Pele:  真搞不懂一匹马是怎么活下来的……

Agent Pele，Agent Lalahon和 Agent Chantico查看整个果园。

Agent Lalahon:  奇怪……

Agent Chantico:  怎么，我觉得他俩待在一起挺好。

Agent Lalahon:  我不是在说 418 和他的马，我是指这个果园。

Agent Lalahon指向一棵树。 Agent Pele看向它，表示它很正常。每个枝杈都生长着大量苹果,它们身上也没有异常。每过几秒钟就有一个苹果从树上落下。苹果树不是扎根在土壤中，而是直接长在岩盐上。

Agent Lalahon:  没有土，直接长在矿物表面。也看不到水源。

Agent Chantico:  也许是那匹马在照料它们？要不然就是……魔法？

Agent Lalahon:  现在看来魔法也不是那么不合理的解释，尤其我们还是通过一个巫毒娃娃进到这里的。

Agent Lalahon耸肩。

Agent Lalahon:  我不知道。Pele，你在录像吗？

Agent Pele:  别担心，Lalahon，我给果园录了像。研究人员会对这些感兴趣的。

Agent Hephaestus靠近了 PoI-418 和 PoI-418-H。

Agent Hephaestus:  所以这就是她了？

PoI-418抬头看着 Agent Hephaestus。

PoI-418:  在哪里我都认得她身上的白斑。

PoI-418-H 冲他嘶鸣。 PoI-418笑起来。

PoI-418:  我想她也认得我。

Agent Hephaestus看向倒塌的岩柱，小心地把手放上去。

Agent Hephaestus:  她活这么久真够奇怪的。你说她在你之前就在这里了？

PoI-418:  是啊……树上有那么多苹果给她吃，河那边有水源，还有这么大的地方可以跑动……

Agent Hephaestus:  我听说有时候马会靠吃人活命。也许就是因为这个这里才一个人都没有？

PoI-418局促地笑笑，转向Agent Hephaestus给了他一拳。他没有命中，反而打在柱子上。柱子发出响亮的嘎吱声，像是即将崩塌得更厉害。

PoI-418:  闭嘴，那只是个谣言。

大块岩盐从柱子上崩落下来，直接落向 Agent Hephaestus，PoI-418和 PoI-418-H。

Agent Hephaestus:  听着，你回去之后，趁着记忆还没被清除，去读读那个俄国士兵被马吃了的故事，听上去很疯狂，但挺有意思的。

Agent Pele:  418，Hephaestus，赶紧从那里离开！

Agent Hephaestus看向上方，快速从所站的地方闪开。PoI-418迷惑地看着 Agent Pele。

PoI-418:  等等，发生了什么？有什么——

几大块岩盐落到 PoI-418和 PoI-418-H头上。PoI-418立刻被打中，然后就消失在原地。PoI-418-H不见踪影。

Agent Hephaestus:  该死！

Agent Chantico:  呃，至少他们出去之后能联系上。这就说明Sydneigh被成功提取了，对吗？

PoI-418-H嘶鸣一声。

Agent Lalahon:  ……操。

剩下的人靠近断柱搜寻 PoI-418-H。PoI-418-H似乎就在断柱旁边，情绪激动，四肢以扭曲的角度地张开。它不断嘶叫，但不管它怎么动作，那根断柱都纹丝不动。

Agent Hephaestus:  这他妈怎么回事？

Agent Lalahon:  看上去像是某种程序故障……

PoI-418-H僵住了，四条腿水平向两侧伸展。下一刻它又站在断柱了旁边，毫发无伤。它开始朝果园走去，还吃了一个树上落下的苹果。

Agent Pele:  这不像是程序故障，Agent Lalahon。

Agent Lalahon:  你想说什么？

Agent Pele:  你也看到刚刚的怪事了，这只是一个拙劣的程序，写它的人肯定连常识都不知道。空气坏境不符合常识、人可以穿过地面、反常的温度……

Agent Chantico:  ……所以这整个地方就是一个程序？

Agent Lalahon:  那匹马只是它的一部分，对吗？

Agent Pele:  嗯，考虑到它没有和外界进入的人一样死亡然后消失，我想确实是这样。

队员们沉默地看着 PoI-418-H吃掉了更多苹果。

Agent Chantico:  所以……我们要干什么？

Agent Pele:  把这件事报告给基金会。他们会做好需要做的事。

Agent Hephaestus:  那么 418 呢? 我们怎么给他交代？

Agent Pele:  反正他自愿接受了记忆清除。即使我们告诉他他的马压根就没存在过，对他来说也没有区别。

PoI-418-H离开果园走上桥，队伍再次陷入沉默。

Agent Lalahon:  可怜的家伙。

Agent Pele:  本来可能更糟的。无论如何，那只是一匹马。

Agent Lalahon:  是啊，但……他为了找回她经历了那么多。

Agent Pele:  那不重要。他知道这件事的危险性，也知道这次行动不一定会成功。实际上，他因此帮我们更深入地了解了这个地方。

Agent Hephaestus:  别以为他是为了科学研究跟我们来的。

Agent Pele叹口气。

Agent Pele:  我们走吧。回去之后就知道是怎么回事了。

Agent Pele拔枪对准自己的头，然后开枪。

 
附加记录：

机动特遣队Epsilon 9离开SCP-4111后将行动过程中的发现报告给了基金会研究人员。PoI-418先于他们返回后被护送离开测试场所，他未被告知SCP-4111或 PoI-418-H的本质。

PoI-418现居住于基金会监控下的华盛顿州的█████地区。到目前8为止，他完全安全且未受到心怀不轨者的侵扰。
